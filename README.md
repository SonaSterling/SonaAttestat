## 📝 Описание

**Currency Converter** — это графическое приложение для конвертации валют в реальном времени. Программа получает актуальные курсы валют через внешнее API, сохраняет историю конвертаций в JSON и предоставляет удобный интерфейс для работы.

### Возможности:
- ✅ Конвертация между любыми валютами
- ✅ Актуальные курсы через API (exchangerate-api.com)
- ✅ Сохранение истории конвертаций
- ✅ Валидация ввода (только положительные числа)
- ✅ Удобный интерфейс с таблицей истории
- ✅ Очистка истории с подтверждением

## 🔑 Как получить API-ключ

Приложение использует бесплатное открытое API **Exchangerate API**, которое **не требует ключа** для базового использования.

### Опционально (для более стабильной работы):

1. Перейдите на сайт: https://app.exchangerate-api.com/sign-up
2. Зарегистрируйтесь (бесплатный тариф — 1500 запросов/месяц)
3. Скопируйте ваш API-ключ
4. В файле `api_client.py` раскомментируйте строку с `PRIVATE_URL` и вставьте ключ

```python
# Пример настройки ключа:
PRIVATE_URL = "https://v6.exchangerate-api.com/v6/ВАШ_КЛЮЧ/latest/"
