## 📝 Описание проекта

**Currency Converter** — это профессиональное графическое приложение для конвертации валют в реальном времени. Программа получает актуальные курсы валют через внешнее API, сохраняет историю всех операций в JSON и предоставляет интуитивно понятный интерфейс.

### 🎯 Основные возможности:
- ✅ Конвертация между 160+ валютами мира
- ✅ Актуальные курсы через ExchangeRate API
- ✅ Автоматическое сохранение истории в JSON
- ✅ Валидация ввода (только положительные числа)
- ✅ Удобная таблица истории с прокруткой
- ✅ Очистка истории с подтверждением
- ✅ Обработка ошибок API и интернет-соединения
- ✅ Индикация загрузки при запросах

## 🔑 Как получить API-ключ

Приложение использует **бесплатное открытое API** ExchangeRate API, которое **не требует ключа** для базового использования (до 1500 запросов/месяц).

### Альтернативный вариант (более стабильный):

1. Перейдите на сайт: https://app.exchangerate-api.com/sign-up
2. Зарегистрируйтесь (бесплатный тариф — 1500 запросов/месяц)
3. Подтвердите email
4. Скопируйте ваш API-ключ из личного кабинета
5. В файле `api_client.py` раскомментируйте строки с `PRIVATE_URL` и укажите ключ:

```python
